Uma landing page responsiva para o filme "Retrato de Uma Jovem em Chamas", desenvolvida com HTML5, SCSS e JavaScript.

 Características
-  Design responsivo para todos os dispositivos
-  Paleta de cores personalizada (#3C2A47)
-  Navegação suave entre seções
-  Animações CSS modernas
-  Otimizada para performance
-  Código modular com SCSS

 Tecnologias Utilizadas

- HTML5 - Estrutura semântica
- SCSS - Estilização avançada
- JavaScript - Interatividade
- Gulp - Automação de tasks
- Git - Controle de versão
